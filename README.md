# @vited/runner-react-icon

## Install

### Run directly

```bash
npx @vited/runner-react-icon
```

### Install on system

```bash
npm i -g @vited/runner-react-icon
%binName%
```

### Install in project

```bash
npm i -D @vited/runner-react-icon
```

Add script entry:

```json
{
  "scripts": {
    "%binName%": "%binName%"
  }
}
```

Run:

```bash
npm run %binName%
```

## Options

### `--help`

Show help

### `--version`

Show version
